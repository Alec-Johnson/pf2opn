Why does this exist? pfsense -> opnsense is not a straighforward conversion
There have been some rumblings in the homelab (and enterprise) community about the lack of a migration path from pfSense to Opnsense.
This tool is an attempt to provide a solution to that problem. This tool does not guarantee a perfect conversion. It is a best effort attempt to convert your pfSense configuration to Opnsense.
